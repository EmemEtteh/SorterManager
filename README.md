# SorterManager
Runs different sorting arrays and compares the runtime.

The project is actually on the Development branch, so switch to that please.

To run the algorithm, please download the project and run it in Intellij. 
In the command window, some prompts will appear with regaurds to the sort algorithm to run.
Please choose 1 or 2 to choose either the bubble sort or insertion sort respectively. Next,
A prmopt to choose the array length will appear. Please enter yout desired length.
The program will then run and output the unsorted array, algorithm chosen, sorted array and
the time taken.
